<h1>TUVAL</h1>

<h2>Proje Konusu: </h2>

Bu proje sizin el hareketlerinizle ekrana yazı yazar. 

Sadece işaret parmak: Yazı yazma modu

İşaret parmak + orta parmak: Silgi modu

<h2>Projeyi Başlatma: </h2>

Depoyu bilgisayarınıza indirin.

Terminalde proje klasörüne gidin.

Şu komutla sanal ortamı klasör içine kurun: 'conda env create -f environment.yml --prefix ./env'

Ortamı aktif edin: 'conda activate ./env'

Ardından 'python tuval.py' ile çalıştırılabilir.
